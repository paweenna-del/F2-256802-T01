# Rock-Paper-Scissors Game (Java)

โปรแกรมเป่ายิ้งฉุบที่พัฒนาด้วยภาษา Java  
มีระบบคะแนน (Rating), Win Streak Bonus และ Comeback Bonus  
ผู้เล่นสามารถเล่นซ้ำได้เรื่อย ๆ จนกว่าจะพิมพ์ !exit เพื่อออกเกมและสรุปคะแนน

## กติกาการเล่น
- ผู้เล่นต้องเลือก: rock, paper, scissors
- ชนะ: ได้ 3 คะแนน
- เสมอ: ได้ 1 คะแนน
- แพ้: ได้ 0 คะแนน
- ชนะติดต่อกันจะได้โบนัสจาก Win Streak
    โดย 2 Win Streak จะได้บวกเพิ่ม 1 คะแนน
    และ 3 Win Streak ขึ้นไปจะได้บวกเพิ่ม 2 คะแนน
- แพ้ติดต่อกันตั้งแต่ 3 ครั้งขึ้นไป จะได้ Comeback Bonus +2 คะแนน
- หากผู้เล่นพิมพ์ !exit โปรแกรมจะจบการทำงานและสรุปคะแนนทั้งหมด

## วิธีคอมไพล์และรันโปรแกรม

1. เปิด Command Prompt หรือ Terminal
2. เข้าไปยังโฟลเดอร์ที่มีไฟล์ .java
3. คอมไพล์โปรแกรมด้วยคำสั่ง:
    javac RockPaperScissors.java
4. รันโปรแกรมด้วยคำสั่ง:
    java RockPaperScissors
